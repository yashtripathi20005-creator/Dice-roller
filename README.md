# 🎲 Dice Roller with Animation

A feature-rich dice rolling application with smooth animations built with Python and PyQt5.

## Features

- **Multiple Dice Types**: d4, d6, d8, d10, d12, d20, d100
- **Multiple Dice**: Roll 1-10 dice simultaneously
- **Smooth Animations**: Dice bounce and rotate during rolling
- **Visual Display**: See individual dice values
- **History Tracking**: Keep a record of all rolls
- **Clean UI**: Modern, intuitive interface
